# GitHub 文件批量下载器

一个功能强大的浏览器内工具，可方便地从 GitHub 仓库下载文件，支持镜像加速、文件树选择、ZIP 打包、下载历史等。

## 特性

* 支持多种镜像站
* 树形文件列表，可展开/折叠文件夹
* 全选/反选/选择未下载文件
* 打包选中的文件为 ZIP（支持中途打包）
* 记录下载历史，保存下载位置
* 实时下载速度、剩余时间、进度条
* 文件搜索

## 使用方法

1. 在浏览器中打开 `gitload.html`。
2. 输入 GitHub 仓库地址或手动填写用户名、仓库名、分支。
3. 选择镜像站（部分镜像可能不支持文件树，需先使用官方加载列表）。
4. 点击“加载文件列表”，等待文件树生成。
5. 勾选需要下载的文件或文件夹。
6. 点击“下载所选”直接保存到本地，或“打包为 ZIP”下载压缩包。
7. 如需保存到指定文件夹，可先点击“选择位置”（需浏览器支持 FileSystem API）。

> **注意**：若浏览器不支持文件夹选择，将使用传统下载方式。
